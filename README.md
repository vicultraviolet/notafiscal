# Miliogo Nota Fiscal

Programa que lê QR-Codes de notas fiscais NFC-e e importa os dados dos produtos no https://miliogo.com/cupom/

Eventualmente terá funcionalidade de consultar os produtos pelo app em si, além de gráficos relacionados aos seus gastos


Versão 2.0 no Android do https://github.com/ivantgm/consulta_sat
